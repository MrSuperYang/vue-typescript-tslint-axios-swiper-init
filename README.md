# vue-typescript-axios-swiper-init

基于vue-cli 3.0创建
配置axios请求头参数等，实现请求超时后重新请求，配置在src/http/AxiosClass.ts中。
配置vue-awesome-swiper的描述文件。
配置tslint。
推荐下载 EditorConfig for VS Code插件，保存的时候自动格式化 LF <-> CRLF。
推荐开启 tslint 保存自动fix的功能。
